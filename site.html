Sobre mim

Moro em Portugal mas atendo clientes do mundo todo. Sou conhecido por fazer produtos de qualidade, durabilidade e que agregam valor para meus clientes.

Trabalho usando a web como plataforma, ou seja, respiro HTML5, CSS3 e JavaScript (ou melhor: ECMASCript). Crio sites para todos, seguindo as principais diretivas de acessibilidade, responsividade e web semântica, sem descuidar da qualidade de código.

Como trabalho

Renan é o melhor desenvolvedor front-end com quem já trabalhei. Muito eficiente e muito capaz. Recomendo sem dúvidas!

José Souza, Fiat

Satisfazer meus clientes é prioridade. Para isso, garanto um processo de desenvolvimento altamente interativo, baseado em feedback contínuo. Não trabalho com escopo fechado: o cliente é que decide quando o produto está pronto.

Também não trabalho com prazos fechados: qualidade é importante demais para ser sacrificada.

Resumo

Habilidade Descrição
HTML5 Procuro desenvolver sites com um HTML enxuto, legível e muito semântico, a fim de proporcionar aos meus clientes um site leve, ágil e fácil de ser encontrado em por mecanismos de busca.
CSS3 Considerando um HTML bem semântico, o CSS é bem trabalhado para que o browser carregue somente o necessário de forma rápida e eficiente, assim avitando uma difícil renderização da página, deixando-a mais rápida e proporcionando uma execente experiência ao usuário. Algumas funcionalidades, como preenchimento de formulários e outros, também são feitos basicamente com um HTML semantico e um CSS bem estruturado, tendo em vista que usuários mais experiêntes possam desabilitar o JavaScript.
JavaScript Interações bem organizadas, com um código limpo e bem legível faz com que o site seja de fácil atualização e manutenção.
Git Procuro sempre subir meus projetos ao Github, desde o mais simples ao mais complexo, sempre deixando o código aberto. Alem disso, procuro ser um membro ativo dentro da comunidade front-end, pois acredito fortemente que conhecimento deve ser compartilhado.
